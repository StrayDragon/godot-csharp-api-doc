Relative

The drag position relative to the previous position (position at the last frame).

Note: Relative is automatically scaled according to the content scale factor, which is defined by the project's stretch mode settings. This means touch sensitivity will appear different depending on resolution when using Relative in a script that handles touch aiming. To avoid this, use ScreenRelative instead.

public Vector2 Relative { get; set; }

Property Value

Vector2

ScreenRelative

The unscaled drag position relative to the previous position in screen coordinates (position at the last frame). This position is not scaled according to the content scale factor or calls to XformedBy(Transform2D, Vector2?). This should be preferred over Relative for touch aiming regardless of the project's stretch mode.

public Vector2 ScreenRelative { get; set; }

Property Value

Vector2

ScreenVelocity

The unscaled drag velocity in pixels per second in screen coordinates. This velocity is not scaled according to the content scale factor or calls to XformedBy(Transform2D, Vector2?). This should be preferred over Velocity for touch aiming regardless of the project's stretch mode.

public Vector2 ScreenVelocity { get; set; }

Property Value

Vector2

Tilt

Represents the angles of tilt of the pen. Positive X-coordinate value indicates a tilt to the right. Positive Y-coordinate value indicates a tilt toward the user. Ranges from -1.0 to 1.0 for both axes.

public Vector2 Tilt { get; set; }
