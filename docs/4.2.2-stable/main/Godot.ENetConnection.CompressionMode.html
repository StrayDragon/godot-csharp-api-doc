public enum ENetConnection.CompressionMode : long

Fields

Fastlz = 2

FastLZ compression. This option uses less CPU resources compared to Zlib, at the expense of using more bandwidth.

None = 0

No compression. This uses the most bandwidth, but has the upside of requiring the fewest CPU resources. This option may also be used to make network debugging using tools like Wireshark easier.

RangeCoder = 1

ENet's built-in range encoding. Works well on small packets, but is not the most efficient algorithm on packets larger than 4 KB.

Zlib = 3

Zlib compression. This option uses less bandwidth compared to Fastlz, at the expense of using more CPU resources.

Zstd = 4

Zstandard compression. Note that this algorithm is not very efficient on packets smaller than 4 KB. Therefore, it's recommended to use other compression algorithms in most cases.
