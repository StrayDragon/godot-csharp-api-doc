No collision for particles. Particles will go through GpuParticlesCollision3D nodes.

Hide particles instantly when colliding with a GpuParticlesCollision3D node. This can be combined with a subemitter that uses the Rigid collision mode to "replace" the parent particle with the subemitter on impact.
