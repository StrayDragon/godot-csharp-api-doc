Class TileData

Namespace
Godot
Assembly
GodotSharp.dll

TileData object represents a single tile in a TileSet. It is usually edited using the tileset editor, but it can be modified at runtime using _TileDataRuntimeUpdate(int, Vector2I, TileData).
